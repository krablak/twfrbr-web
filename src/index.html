Errors are opened points directly to location in file.

Bundle assets overview.

Development process stays same as with classic webpack dashboard plugin.

How it works?

Solution is based on modified webpack dashboard plugin which publish all build process events using socket.io. Events contains all information about current build step, progress and also build result including bundle assets structure description.

Within IDEA plugin exist run configuration type that subscribes to webpack dashboard plugin socket.io events. Received events are parsed and transformed into content of plugin UI.
